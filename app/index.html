Introduction


This app is mainly targeted at those interested in investing in the movie industry, more specifically in movie distributors.
Providing charts that help visualize:
--Trends over the years regarding genres across the years.
--The various' distributors strong and weak points
--How efficient each distributor is with money.
--Which genres are more profitable for a given target audience.
As well as some simple data exploration features to make the user more acquainted with the dataset.
In order to help the user make a more informed decision on which distributor to invest in.
Though this data is relevant for those interested in the movie industry regardless of whether they're investing or not.
